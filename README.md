# ***Informe sobre Trabajo Práctico Integrador ***
## De wireframes a sitio web real

### pasos a seguir:

- 1: Tener en cuenta los wireframes anteriormente realizados para adaptarlos a un diseño responsivo. 
- 2: Establecer el nombre de la pagina (Redes).
- 3: Agregar las pestañas de los wireframes como tarjetas separadas. 
- 4: A cada tarjeta realizada se le va a : agregar titulo (32px) , subtitulos (24px) y textos (18px) a cada pestaña, incluyendo su tipografia (Arial, sans-serif). Estos estan basados en los wireframes.
- 5: Adaptar el tamaño de las tarjetas para un tamaño responsivo entre computador (1920x1080 px) y celular (780X1080 px).
- 6: asignar paleta de colores a cada tarjeta adoptando el estilo anteriormente elejido en los wireframes. se utiliza una transicion de los colores para que correspondan a la paleta de colores original de una manera natural. Ademas de ser un diseño mas responsivo.
