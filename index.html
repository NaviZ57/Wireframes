Las Redes

Son un conjunto de dispositivos interconectados que se comunican a traves de cables o internet

Redes LAN & WAN

LAN (Local Area Network):

Conectan dispositivos en un area local utilizando un router o un switch para conectar los distintos dispositivos

WAN (Wide Area Network):

Conecta dispositivos en un area geografica mayor utilizando multiples routers interconectados para pasar informacion

Protocolos de Comunicacion

Son las reglas que utilizan los dispositivos para su comunicacion

TCP(Protocolo de Control de Transmision) garantiza la entrega confiable de paquetes de datos

IP(Protocolo de Internet) se encarga de direccionar y encaminar dichos paquetes a traves de las redes

Ethernet: protocolo usado por los cables de red para conectar una red en un area local

Dispositivos de Red

Router

Su función es la de establecer la mejor ruta que destinara a cada paquete de datos para llegar a la red y al dispositivo de destino a partir de una IP

Switch

Dispositivo que sirve para interconectar diferentes equipos y crear una red de área local.

También permite centralizar la gestion de diferentes puntos de acceso inalámbrico.
